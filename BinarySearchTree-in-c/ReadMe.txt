## Contents

ReadMe.txt ------------------- readme file for BST
main.c ----------------------- Contains main function that tests the bst
bst.c ------------------------ Contains BST functions
bst.h ------------------------ Contains BST function declarations from bst.c, and "Tree" and "TreeNode" structs
extra.c ---------------------- Contains immitation of strcmp function
extra.h ---------------------- Contains declaration of immitation of strcmp function from extra.c and "example" struct for testing
Makefile --------------------- makefile for running the main function to test bst
